Методы массивов методы либо изменяют массив, либо возвращают новый.
  • concat - копирует ссылки на объекты в новый массив. возвращает НОВЫЙ массив. не меняет исходные массивы. array.concat( secondArray, thirdArray, ..., xArray )
  • copyWithin - copies array elements to another position in the array. overwrites. does not add items. array.copyWithin(target, start, end) [0,1,2,3].copyWithin(0,2,3) // [ 2, 1, 2, 3 ] [0,1,2,3].copyWithin(0,2) // [ 2, 3, 2, 3 ] [0,1,2,3].copyWithin(-1,2) // [ 0, 1, 2, 2 ]
  • entries - returns a new Array Iterator object that contains the key/value pairs for each index in the array. does not change the original array. array.entries() ['a', 'b', 'c'].entries().next().value // [0, 'a']
  • every - true if the callbackFn function returns a truthy value for every array element. Otherwise, false. array.every(function(currentValue, index, arr), thisValue) [0,1,2].every(e=>e>0) // true
  • fill - fills specified elements in an array with a value. overwrites the original array. array.fill(value, start, end) [0,1,2,3,4].fill(5,1,2) // [ 0, 5, 2, 3, 4 ] [0,1,2,3,4].fill(5,1,3) // [ 0, 5, 5, 3, 4 ] [0,1,2,3,4].fill(5,1) // [ 0, 5, 5, 5, 5 ]
  • filter - creates a new array filled with elements that pass a test provided by a function. does not change the original array. array.filter(function(currentValue, index, arr), thisValue)
  • find - returns the value of the first element that passes a test. array.find(function(currentValue, index, arr),thisValue) [5, 12, 8, 130, 44].find(e=>e>10) // 12
  • findIndex - returns the index (position) of the first element that passes a test OR -1. does not change the original array. array.findIndex(function(currentValue, index, arr), thisValue) [5, 12, 8, 130, 44].findIndex(e=>e>13) // 3
  • flat
  • flatMap - A new array with each element being the result of the callback function and flattened to a depth of 1. array.flatMap((currentValue, index, array) => {} ) [0,1,2].flatMap(e=>[e, e+1]) // Array(6) [ 0, 1, 1, 2, 2, 3 ]
  • forEach - calls a function for each element in an array. method is not executed for empty elements. array.forEach(testfunc(currentValue, index, arr), thisValue) [0,1,2].forEach(item => {console.log(item+1)}) // 1 2 3
  • from - returns an array from any iterable object. use it as Array.from(). Array.from(object, mapFunction, thisValue) [1,2,3].from() // undefined. use it as Array.from() Array.from([1, 2, 3]) // [1,2,3] Array.from([1, 2, 3], e=>e+1) // [2,3,4] Array.from("text") // [ "t", "e", "x", "t" ]

  • isArray - use it as Array.isArray(). returns true if an object is an array, otherwise false. Array.isArray(obj) [0,1,2].isArray() vs Array.isArray([0,1,2]) // undefined vs true
  • join - объединяет все элементы массива (или массивоподобного объекта) в строку. [0,1,2].join() // 0,1,2

  • map - creates a new array from calling a function for every array element. does not execute the function for empty elements. does not change the original array. array.map(testfunc(currentValue, index, arr), thisValue) [0,1,2].map(e=>{return e+2}) // [2,3,4]
  • of - A new Array that contains the values that you pass to it. Array.of(element0[, element1[, ...[, elementN]]]) // method creates an array of one number. Array(3) vs Array.of(3) // [undefined, undefined, undefined] vs [3]

содержит подробности о том, что произошло.
получить объект Event:
elem.addEventListener('click', function(event) {
console.log(event);
});
или
elem.onclick = function(event) {
console.log(event);
});
или
button onclick="alert(event.type)"
некоторые свойства объекта event:
event.type event.currentTarget удалить соответствующий обработчик: button.onclick = null. element.removeEventListener(event, handler[, options]);

function foo(event) {console.log(event.clientX);} function foo(event) {console.log(event.offsetX);} function foo(event) {console.log(event.pageX);} function foo(event) {console.log(event.screenX);}
