Свойства и методы массива
  • свойства хранят сведения о массиве.
  • методы либо изменяют массив, либо возвращают новый.
перечень свойств массивов array.length // the length - returns or sets the number of elements in a given array arr[arr.length - 1] // the last number of elements in a given array - returns the last number of elements in a given array перечень методов массивов array.concat( secondArray, thirdArray, ..., xArray ) // возвращает НОВЫЙ массив. не меняет исходные массивы. - копирует ссылки на объекты в новый массив array.indexOf(searchElement[, fromIndex = 0]) - возвращает первый индекс, по которому данный элемент может быть найден в массиве или -1, если такого индекса нет. [0,1,2].join() // 0,1,2
[0,1,2].join("") // 012
[0,1,2].join("-") // 0-1-2 - объединяет все элементы массива (или массивоподобного объекта) в строку. array.pop( ) // the removed last element - Последний элемент будет удален из массива array.push( element1, element2, ..., elementX ) // the new length - Один или несколько элементов будут добавлены в конец массива array.shift( ) // the removed first element - Первый элемент будет удален из массива array.unshift( element1, element2, ..., elementX ) // the new length - Один или несколько элементов будут добавлены в начало массива

RegExp
создать регулярное выражение : let reg = new RegExp("template", "g"); // через конструктор let regexp = /template/g; // через слэш флаги : i g m s u y str.match() let result = str.match(template/g); // поиск "template" в "str"
При отсутствии совпадений возвращается не пустой массив, а именно null.
чтобы результатом всегда был массив :
let result = str.match(/template/g) || [ ];
Задача:
найти в тексте число ХХХ (цыфры не известны)
"a number is 123".match(/\d\d\d/); // Array [ "123" ]
Задача:
найти в тексте первые три символа, не являющихся цыфрами (символы не известны)
"a number is 123".match(/\D\D\D/); // Array [ "a n" ]
Задача:
найти в тексте "символ символ пробел"
"a number is 123".match(/\D\D\s/); // Array [ "er " ]
Задача:
найти в тексте "пробел символ НЕпробел"
"a number is 123".match(/\s\D\S/); // Array [ " nu" ]
