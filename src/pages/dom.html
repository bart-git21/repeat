Class || Style
Классы – всегда предпочтительный вариант по сравнению со style.
Используй свойство style только, если вычисляешь свойства элемента динамически.
elem.className
соответствует атрибуту "class"
elem.className; elem.classList
это объект с методами для добавления/удаления одного класса.
elem.classList.add/remove("class_name") elem.classList.toggle("class_name") elem.classList.contains("class_name")
Задача:
перечислить все классы при помощи for..of
for (let name of elem.classList) {alert(name);}
elem.style
это объект, который соответствует тому, что написано в атрибуте "style"
elem.style.borderLeftWidth elem.style.cssText
это свойство удаляет все существующие стили
elem.style.cssText=`color: red;
width: 100px;
text-align: center;`; getComputedStyle
это метод, чтобы получить стиль элемента.
Node! нужно указать точное свойство. Например: paddingLeft.
getComputedStyle(element, [pseudo])
pseudo Указывается, если нужен стиль псевдоэлемента, например ::before.
let computedStyle = getComputedStyle(document.body);
alert( computedStyle.marginTop );
