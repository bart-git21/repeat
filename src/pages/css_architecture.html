Блок
Элемент
Элементы не имеют смысла вне блока. Классически отделяются двумя подчеркиваниями:div class = "block-name__elem". разделитель можно использовать другой.
Модификатор
Определяет внешний вид, состояние и поведение блока либо элемента. В классической схеме имя модификатора отделяется одним подчеркиванием.
Принципы написания CSS
  • не использовать комбинированные селекторы вида .class1.class2 { }
  • не использовать тег и класс в селекторе h1.class { }
  • не использовать атрибуты тегов input[type=submit] { }
Преимущства
  • БЭМ избавляет от вложенности селекторов header nav button a { }
  • Можно миксовать стили универсальных блоков и дополнительных классов form class = "form search"
  • Файловая структура проекта облегчает перенос блока из проекта в проект. Чтобы вместе с версткой перенести все стили и поведение блока, достаточно скопировать папку этого блока в новый проект.

5 составляющих
  • Base - правила по умолчанию. Например, для html, body, a, a:hover и т.д.
  • Layout - стили, с помощью которых страница разделяется на секции.
  • Module - стили модулей (блоков), которые могут использоваться несколько раз на одной странице
  • State - внешний вид контента в определенном состоянии (видимый/скрытый, светлый/темный)
  • Theme - похоже на State
