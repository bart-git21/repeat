columns: column-count & column-width
Node! в column-width % не применяются
column-width: 100px; // N колонок шириной 100px или шире columns: auto 100px; // N колонок шириной 100px или шире columns: 3 400px; // 3 колонки, ширина 400px или шире; если ширины контейнера на 3 колонки не хватит - ширина каждой 400 или шире, но количество уменьшится column-count: 3; // 3 колонки N ширины columns: 3; // 3 колонки N ширины
если после расчета количества колонок и их ширины останется пустая ширина, куда не поместится еще 1 колонка, все колонки растянутся, заполнив пустоту.

img
object-fit
Свойство object-fit определяет, как содержимое элемента, должно заполнять контейнер относительно его высоты и ширины.
fill - заполнить всю область внутри блока
contain - "помещённый внутрь" блока, ограничиваясь его шириной и высотой.
cover - "покрытие" блока, ограничиваясь его шириной и высотой.
none- Смещаемый контент не изменяет свой размер
scale-down - Контент изменяет размер, сравнивая разницу между none и contain, для того, чтобы найти наименьший конкретный размер объекта.
